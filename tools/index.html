使用教程

⚠️ 前提条件
这些工具只能分析由 AGP (ag-pulse) 导出的 JSON 文件。
先将 AGP 项目拉取到本地（git clone + pnpm install），然后在 AGP 目录下运行导出命令。
🔒 安全提示
导出的对话数据（.agp/）包含完整对话内容，请务必将其加入 .gitignore 防止泄露！
1
确保 Antigravity 正在运行
在 Antigravity 中打开一个工作区，确保 Antigravity 处于活跃状态。
2
导出对话数据
在 AGP 项目目录下执行，指定要导出的工作区路径：
# 导出指定工作区的对话（交互式选择）
pnpm export C:\path\to\workspace

# 导出指定工作区的所有对话
pnpm export-all C:\path\to\workspace

# 不传路径 = 导出 AGP 自身目录的对话
pnpm export
JSON 文件会保存在 <目标工作区>/.agp/exports/ 目录下
3
拖入 JSON 文件
打开上面任一工具页面，将导出的 .json 文件拖入即可。所有解析均在浏览器本地完成，数据不会上传。
